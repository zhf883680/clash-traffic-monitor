# v2.1.3

## Highlights

- 连接设置改为弹窗
  - 顶部“连接设置”按钮现在打开独立弹窗
  - 保存逻辑保持不变，首次必须配置时仍会优先引导连接 Mihomo
  - 支持遮罩点击关闭、右上角关闭按钮和 `Esc` 快捷关闭
- 自动切换配置改为弹窗
  - 页面正文不再直接铺开自动切换配置区
  - 顶部“自动切换”按钮打开完整配置与最近触发记录
  - 刷新策略组、保存配置、取消等现有行为保持不变
- 当前看板上下文新增自动切换状态模块
  - 在原有 3 个摘要模块基础上新增第 4 个模块“自动切换”
  - 直接展示自动切换是否开启
  - 开启后补充阈值、冷却时间、启用策略组数量，以及自动恢复状态等基础信息

## Behavior

- 这次更新主要调整前端信息层级与交互结构，不改变自动切换和连接设置的后端接口语义。
- 如果未配置 Mihomo，页面仍会优先弹出连接设置并阻止主看板继续加载。
- 自动切换的摘要状态会随配置加载、保存和策略组刷新即时更新。
