# v2.0.0

## Highlights

- 新增“大流量自动切换”能力
  - 按 1 分钟窗口统计单个域名流量
  - 任意域名超过阈值后，自动执行预设策略组切换
  - 支持冷却时间，避免频繁重复切换
- 新增 Mihomo 策略组读取与切换控制
  - 仅展示并控制 `select` 和 `fallback`
  - 每个策略组可单独指定目标节点
  - 记录最近触发事件和执行结果
- 页面交互升级
  - 自动切换面板改为独立入口，默认收起
  - 顶部工具栏布局压缩，给时间选择和操作按钮腾出更多空间
  - 保存或取消后可直接收起自动切换面板

## Notes

- 第一版自动切换不维护域名白名单，也不会自动回切。
- 自动切换只负责执行你预设的策略组切换动作，不接管 Mihomo 规则判断逻辑。
- 建议先从一个托管组开始，例如 `⏬ 大流量节点`，确认行为符合预期后再逐步扩大范围。
