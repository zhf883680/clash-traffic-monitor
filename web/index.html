连接 Mihomo

首次使用前先填写 Mihomo Controller 地址和 Secret。保存后会立刻用于后续采集，并持久化到本地数据库。

  • Docker 升级时如果环境变量里已有值，服务启动后会直接使用并保存。
  • 后续在这里修改连接参数，无需重启页面。
  • 如果 Mihomo 没有设置密钥，Secret 可以留空。
启动时如果传入了 `MIHOMO_URL` 或 `MIHOMO_SECRET`，重启后仍会优先使用环境变量，并覆盖这里保存的值。
